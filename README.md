## 📋 Descripción

Colección de ejercicios prácticos de **manipulación de arrays en C#**. Este repositorio documenta el aprendizaje de estructuras de datos lineales: recorridos, búsquedas, ordenamientos e inserciones. Es la base para estructuras de datos más complejas.

---

## 📚 Ejercicios Incluidos

| # | Ejercicio | Descripción |
|---|---|---|
| 01 | **Carga y recorrido** | Llenar un array con datos del usuario e imprimirlo |
| 02 | **Búsqueda lineal** | Encontrar un elemento dentro del array |
| 03 | **Máximo y mínimo** | Recorrer el array para encontrar el valor más alto y más bajo |
| 04 | **Suma y promedio** | Calcular la suma total y el promedio de los elementos |
| 05 | **Arrays paralelos** | Usar dos arrays con el mismo índice para relacionar datos |
| 06 | **Ordenamiento** | Organizar los elementos de menor a mayor |

---

## 🧠 Conceptos Aplicados

- **Declaración e inicialización** de arrays en C#
- **Ciclos `for`** para recorrido y manipulación
- **Arrays paralelos** para relacionar conjuntos de datos
- **Búsqueda lineal** — comparación elemento a elemento
- **Algoritmos de ordenamiento** básicos (burbuja)
- **Acumuladores y contadores** para estadísticas

---

## 💡 ¿Por qué son importantes los arrays?

Los arrays son la estructura de datos más fundamental en programación. Dominarlos es el primer paso para entender:

```
Arrays  →  Listas  →  Pilas / Colas  →  Árboles  →  Grafos
```

Cada ejercicio de este repositorio construye la intuición necesaria para estructuras más complejas.
